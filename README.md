# 🥗 Body Composition & Macro System

A premium, **100% private**, and **offline-first** PWA for calculating body composition and nutritional requirements. Built for high-performance individuals who value data privacy and efficiency.

## 🚀 Live Access
**Install the App here:** [https://shaanahmed.github.io/bmi-macro-calculator/](https://shaanahmed.github.io/bmi-macro-calculator/)

---

## ✨ Key Features
* **100% Private:** No data ever leaves your device. All calculations happen locally.
* **Offline-First:** Install it as a PWA on your iPhone, Android, or Desktop. Works perfectly without an internet connection.
* **Professional Accuracy:** Advanced formulas for BMI, Body Fat %, and personalized Macro-nutrient targets.
* **Clean UX:** No ads, no tracking, no bloat. Just the data you need.

## 📱 Installation Guide

### Mobile (iOS/Android)
1. Open the [Live Link](https://shaanahmed.github.io/bmi-macro-calculator/) in Safari or Chrome.
2. Tap **'Share'** (iOS) or the **three dots** (Android).
3. Select **'Add to Home Screen'**.

### Desktop
1. Open the link in Chrome or Edge.
2. Click the **Install Icon** in the right side of the address bar.

---

## 🛠 Tech Stack
* **Framework:** React 18 + TypeScript
* **Bundler:** Vite
* **PWA:** vite-plugin-pwa (Service Workers & Manifest)
* **Styling:** Tailwind CSS
